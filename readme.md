## About

A browser-based Tetris game built with vanilla JavaScript and HTML5 Canvas. Features a sleek dark neon design with glowing cyan borders, classic Tetris mechanics, score tracking, and local high score persistence.

## Controls

| Key | Action |
|:---:|:-------|
| `Left` | Move left |
| `Right` | Move right |
| `Up` | Rotate |
| `Down` | Soft drop |

## Features

- All 7 classic tetromino shapes
- Smooth movement and rotation
- Score counter + persistent best result (survives page reload!)
- Neon-glow visual style (dark background + cyan accents)
- Instant restart button

## Tech Stack

| | |
|:--|:--|
| **Rendering** | HTML5 Canvas (320x640) |
| **Logic** | Vanilla JavaScript |
| **Styling** | CSS3 with neon glow effects |

## Project Structure
